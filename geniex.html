How to install GenieX

Step 1   Connect the genie device to cpus back side
Step 2   Connet your field USB in CPU
Step 3   Start installation based on the system bit 32/64

1-install redmon19 as admin
2-install cutewriter as admin
3-install gs923.exe as admin
4-install gs922.exe as admin
5-install gxprint1.0 as admin

Step 4   Open device and printers


1-Cutepdf printer properties-- select PORT as RPT1 redirected port.
2-Set the paper size as 80 width and 297 height in mm
3-set Cutepdf as default printer


Step 5 install nxlog and ushopgeniexL 3.4.6
step 2   activate test mode

you can able to see 4 options

1-multiprint-genei name as generic text only printer name as per physical printer name
2-printer (swap printer if required)
3-windows title name------need to find with scan detection tool
4-anydesk path---set detais.
5-Deactivate Testmode
