How to install GenieXL

Step 1   Connet your field USB in CPU
Step 2   Start installation based on the system bit 32/64

1-install redmon19 as admin
2-install cutewriter as admin
3-install gs923.exe as admin
4-install gs922.exe as admin
5-install gxprint1.0 as admin



Step 3   Open device and printers

>

1-Cutepdf printer properties-- select PORT as RPT1 redirected port.
2-Set the paper size as 80 width and 297 height in millimeter
3-set Cutepdf as default printer


Step 4 open 0.5.2 folder and Run utils (registeerdevice) as admin based on the 32/64
Step 5 copy the pos id e0:5a:1f:8c:0d:a8 and post in backend group.
(wait for the confirmation from backend.they will assign one device id to you)
After getting confirmation from backend run that utils again and get confirmation from pc

Step 6 install RS genie xl aplication.

Step 7 install nxlog and ushopgeniexL 3.4.6
Step 8   activate test mode

you can able to see 4 options

1-multiprint-genei name as gxl:// printer name as per physical printer name
2-printer (swap printer if required)
3-windows title name------need to find with scan detection tool(scanner1.exe)
4-anydesk path---set detais.
5-Deactivate Testmode




Now scan and print then check health...if you need help raise a ticket and post in the group.
