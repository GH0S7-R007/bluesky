How to install Indic OCR for XL

1. Install Tesseract (Run as Admin) according to OS bit version (32/64 bit).


2. Extract utilities according to OS bit version (32/64 bit).


3. Open the utilities32/64 folder extracted > copy sub folder "utilities" and paste under "C:\Program Files\RdSkyTech\gxprint"


4. Extract tessdata-4.0.0-fast and copy entire content (Ctrl=A) and paste under "C:\Program Files\Tesseract-OCR\tessdata" overwriting existing Script folder.
